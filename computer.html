Werking van een Computer

1. Invoer (Input) Je geeft de computer informatie. Dit kan met een toetsenbord, muis, of een andere invoer. 2. Verwerking (Processing) De computer denkt na over wat je hebt ingevoerd. De CPU (centrale verwerkingseenheid) doet dit denkwerk. 3. Uitvoer (Output) De computer laat je zien wat het heeft gedaan. Dit gebeurt op een scherm, een printer, of via geluid uit de luidsprekers. 4. Opslag (Storage) De computer bewaart informatie zodat je het later kunt gebruiken. Dit gebeurt op de harde schijf, een SSD, of een USB-stick. Voorbeeld: Invoer: Je typt een zin met het toetsenbord. Verwerking: De computer verwerkt de zin. Uitvoer: De zin verschijnt op het scherm. Opslag: Je slaat het document op, zodat je het later weer kunt openen.
